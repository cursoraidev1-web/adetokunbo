## Future Image Updates

### How to Change Images:

**1. Find New Images:**
- Visit https://unsplash.com
- Search for: "Black professional", "African business", "Nigerian professional", etc.
- Select an image you like
- Click "Share" → Copy the image URL

**2. Format the URL:**
Add optimization parameters to the URL:
```
https://images.unsplash.com/photo-[ID]?w=800&h=600&fit=crop&q=80
```

**3. Update the Code:**
- **Service Images:** Edit `pages/HomePage.tsx` (lines 11-30)
- **Testimonial Images:** Edit `pages/HomePage.tsx` (lines 34-54)
- **About Images:** Edit `pages/AboutPage.tsx` (lines 55, 110)

**4. Recommended Sizes:**
- **Hero images:** `w=1200&h=900`
- **Service cards:** `w=800&h=600`
